// # 首先使用OpenSSL创建自签名证书：
// ## genrsa命令 生成私钥key文件
//      openssl genrsa 1024 > RSAprivate.key
// 
// ## req命令 通过私钥文件生成请求证书（req==request，CSR）文件（包含信息：私钥+公钥+个人信息+数字签名） "-new"表示新生成一个新的证书请求文件 "-key"指定私钥文件，"-out"指定输出文件，
//      openssl req -new -key RSAprivate.key -out Req.key
//      会提示输入个人信息，必填的信息是域名。因为常规情况下不能通过https+ip访问https申请的证书是和域名进行绑定的，直接使用https+ip地址证书就会校验失败，无法通过https+ip地址访问后台服务。
//      查看创建好的请求证书文件openssl req -in Req.key -text
// 
// ## 通过私钥文件和CSR证书签名生成自签名证书文件（原本应该将CSR证书发给才能得到被权威机构数字签名的证书）
//      openssl x509 -req -days 365 -in Req.key -signkey RSAprivate.key -out singedCrt.key
